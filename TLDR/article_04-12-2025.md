 CRITICAL SECURITY VULNERABILITY IN REACT SERVER COMPONENTS (4 MINUTE
READ) [15] 

 There is a security vulnerability in React Server Components that
allows unauthenticated remote code execution. It exploits a flaw in
how React decodes payloads sent to React Server Function endpoints.
Apps may still be vulnerable even if they don't implement any React
Server Function endpoints. The vulnerability is present in versions
19.0, 19.1.0, 19.1.1, and 19.2.0 of react-server-dom-webpack,
react-server-dom-parcel, and react-server-dom-turbopack.

 EVERYONE IN SEATTLE HATES AI (6 MINUTE READ) [18] 

 Tech workers in Seattle tend to react negatively when the topic of AI
is brought up. This is due to several reasons, like layoffs due to AI,
the proliferation of AI in software, and being forced to use AI tools
whether they work or not. The result is that engineers are no longer
trying because they think they can't, and companies aren't empowering
them because they assume they shouldn't. Bad products are only
reinforcing the belief that AI is doomed.
