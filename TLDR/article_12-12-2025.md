 COMPOUND ENGINEERING: HOW EVERY CODES WITH AGENTS (13 MINUTE READ)
[13] 

 In traditional engineering, each feature makes the next feature
harder to build because of new edge cases, interdependencies, and
issues. In compound engineering, each feature makes the next feature
easier to build because compound engineering creates a learning loop
for agents and team members so that each bug, failed test, or insight
gets documented and used by future agents. AI's knowledge of the
codebase grows along with it, making future development work faster.
This post provides a high-level sense of how to start compound
engineering with AI. 

 AN SVG IS ALL YOU NEED (4 MINUTE READ) [14] 

 SVGs are vector graphics in a simple XML format. They are supported
on almost every device and platform. With embeddable scripts to make
them interactive, SVGs are way more capable than many people realize.
It is entirely feasible to implement an entire data analysis pipeline
in an SVG. A completely self-contained SVG file can fetch data from a
versioned repository, process that data, and generate visualizations
without any server-side magic required.
