 I MADE A TERMINAL PAGER (11 MINUTE READ) [6] 

 Developing Terminal User Interfaces like `kl` for Kubernetes logs
showed the need for better text navigation within these applications.
To address this, a reusable `viewport` component was developed in Go,
designed to handle features such as scrolling, wrapping, searching,

provides an intuitive way to interact with multi-page text.

 CODEX HACKED A SAMSUNG TV (13 MINUTE READ) [16] 

 OpenAI's Codex successfully escalated privileges to gain root access
on a Samsung Smart TV. Researchers provided Codex with an initial
browser shell foothold and the TV's firmware source, tasking the AI
with finding a vulnerability to achieve root. Codex identified a
critical flaw in the `ntksys` kernel driver, which allowed it to map
arbitrary physical memory into user space and overwrite the browser
process's kernel credentials.
