ARTICLES & TUTORIALS

 DON'T RUN SQL MIGRATIONS IN TESTS: HOW I SPED UP THE TEST SUITE BY 2X
(12 MINUTE READ) [8] 

 Large Go test suites often face slowdowns when thousands of SQL
migrations are applied serially for every individual test. To address
this, a content-addressing system identifies a "golden" database file
by hashing the contents of all required migration scripts. If a
matching database already exists, the system clones it instantly using
a backup API, whereas missing versions are lazily generated and cached
for subsequent runs. 

 BRITISH COLUMBIA, TIME ZONES, AND POSTGRES (8 MINUTE READ) [9] 

 British Columbia's shift to a permanent UTC-7 offset in 2026 shows
the risks of storing future appointments in UTC-based database
columns, which can lead to unexpected one-hour shifts in local time.
These columns convert timestamps based on the rules available at the
time of entry, so subsequent updates to time zone data can cause the
stored UTC values to no longer match the user's intended wall-clock
time. 

 SVGS AND PDFS CAN BOTH BE INTERACTIVE (12 MINUTE READ) [10] 

 Both SVG and PDF file formats have underused specs that allow for the
integration of interactive layers and programmable scripts directly
within a document. While SVGs can host self-contained applications
using embedded JavaScript, PDFs use a specialized API that often
struggles with compatibility across different viewing software.

OPINIONS & ADVICE

 MYTHOS AND FABLE CAN MAKE US ALL SAFER. SHUTTING THEM DOWN IS
RECKLESS (7 MINUTE READ) [11] 

 The United States government recently used export-control authority
to force the shutdown of powerful AI models like Mythos 5. This
unilateral action is considered reckless by many security experts
because these models are necessary tools for defenders to identify and
patch vulnerabilities in global software infrastructure. Since
comparable AI capabilities are already being released by international
labs, domestic shutdowns do not eliminate risks but instead deprive
local developers of the resources needed to counter emerging threats. 

 STOP USING JWTS (3 MINUTE READ) [12] 

 JSON Web Tokens should be avoided for managing user sessions because
they are insecure and were never intended for long-term
authentication. Traditional cookie sessions are a more reliable
alternative, with better flexibility and protection against common
security vulnerabilities. Truly stateless authentication is largely
impractical, as secure token management usually requires a persistent
data store to handle revocations and expiration properly. 

 THE WEB WE KNOW IS GOING TO DISAPPEAR (16 MINUTE READ) [13] 

 The evolution of technology has seen information move from physical
disks and bulletin boards to the open Web and mobile apps, and it is
now shifting toward centralized AI interfaces. This new model replaces
traditional search-and-click habits with direct, conversational
answers. As AI becomes the primary gatekeeper, websites are
transitioning from being human destinations to serving as background
infrastructure that feeds data into machine learning models.

 WHY IS META DESTROYING ITS ENGINEERING ORGANIZATION? (18 MINUTE READ)
[18] 

 Meta has quickly dismantled its historically strong engineering
culture by aggressively prioritizing AI initiatives, such as forcing
engineers into data-labeling work, tracking employee activity, tying
performance reviews to AI usage, and reducing autonomy that once made
the company attractive. These changes have led to low morale, talent
flight, organizational chaos, and even major security and reliability
failures, including a high-profile Instagram account takeover incident
allegedly linked to AI-generated and insufficiently reviewed code.
