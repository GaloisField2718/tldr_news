 PERFORMANCE HINTS (40 MINUTE READ) [9] 

 Jeff Dean and Sanjay Ghemawat, some of Google's most important
engineers for over 25 years, wrote a guide on optimizing software
performance. Performance needs to be considered early in critical code
paths instead of solely relying on post-development profiling for
optimization. The guide also covers algorithmic improvements,
efficient memory representation, reducing allocations, avoiding
unnecessary work, and using parallelization. 

 INTRODUCING CSS GRID LANES (10 MINUTE READ) [10] 

 CSS Grid Lanes is a new web standard to natively create flexible,
dynamic masonry-style layouts on the web without JavaScript. This
feature uses `display: grid-lanes` along with existing CSS Grid
properties like `grid-template-columns` or `grid-template-rows` to
define the layout's structure. It supports advanced capabilities such
as variable lane sizes, item spanning, explicit placement, and
introduces a "tolerance" property for fine-tuning item packing order.

OPINIONS & ADVICE

 CODE REVIEW AS HUMAN ALIGNMENT, IN THE ERA OF LLMS (3 MINUTE READ)
[11] 

 Code review within teams should change to be primarily a human
alignment mechanism. Large language models have no memory and have to
rediscover everything in the system from first principles every time
they are run. Humans can collectively maintain a shared vision of what
the system should do. Code review should reconfigure itself around
this purpose. 

 MORE DATABASES SHOULD BE SINGLE-THREADED (8 MINUTE READ) [12] 

 From a DX perspective, every transaction within a shard would always
be serializable and never needs to be retried, making it much easier
to think about. A single-threaded database would be fully horizontally
scalable and have extremely high per-thread throughput, as it wouldn't
need to sync with other threads. It would be a much more predictable
system. While more restrictive, there would be no more hidden race
conditions.

 2025 LLM YEAR IN REVIEW (9 MINUTE READ) [19] 

 2025 was an eventful year for LLMs, with paradigm shifts that
redefined the field. Reinforcement Learning from Verifiable Rewards
(RLVR) came up as a new training stage, allowing LLMs to develop
reasoning strategies and driving much of the year's capability
progress. The year also saw the rise of new application layers, local
AI agents like Claude Code, "vibe coding" that democratized
programming, and early hints of a visual LLM GUI. 

 BACKING UP SPOTIFY (26 MINUTE READ) [20] 

 Anna's Archive has created the world's first preservation archive for
music by backing up Spotify's metadata and music files. This massive
300TB collection covers 256 million tracks and 186 million unique
ISRCs, representing approximately 99.6% of all Spotify listens. The
project archives a "long tail" of less popular music and makes the
entire archive openly mirrorable.
