 HTML'S BEST KEPT SECRET: THE OUTPUT TAG (6 MINUTE READ) [5] 

 The often overlooked HTML `output` tag provides built-in assistive
technology support for dynamic results. Unlike `div` elements with
ARIA attributes, `output` automatically announces its value changes to
screen readers. It can be used with or without a `form`, linking to
input elements via the `for` attribute, and has great browser support.

 BUN 1.3 (68 MINUTE READ) [9] 

 Bun 1.3 is a huge update that turns Bun into a full-stack JavaScript
runtime. Bun now includes a full-stack dev server with hot reloading,
built-in clients for MySQL, Redis, and improved HTTP ergonomics. The
update also introduces new testing and debugging tools like VS Code
integration, concurrent tests, and type testing, as well as better
security features such as Bun.secrets for encrypted credential storage
and CSRF protection. 

 RIFT (GITHUB REPO) [10] 

 Rift is a tiling window manager for macOS with multiple layout
styles, a menu bar icon, mission control, and hot-reloadable
configuration. It allows users to tile windows in i3/sway-like or
BSPWM-like arrangements, provides performant animations, and supports
trackpad gestures for workspace switching.
