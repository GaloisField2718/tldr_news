 HOW NOT TO ANSWER THE SALARY QUESTION (12 MINUTE READ) [8] 

 Job seekers should avoid answering the salary question too early in
the interview process. Discussing salary before understanding the full
job responsibilities and benefits is unhelpful for both parties and
can lead to being undervalued. Candidates should deflect the question
by focusing on industry-standard ranges, the comprehensive benefits
package, or the detailed nature of the role itself.

 WEBMCP (WEBSITE) [13] 

 The WebMCP API is a new JavaScript interface that enables web apps to
expose their functionality as "tools" for AI agents. Web developers
can register these tools, which are JavaScript functions with natural
language descriptions and structured schemas, through a ModelContext
interface. This allows various agents, including LLM-based AI
platforms, browser agents, and assistive technologies, to invoke
client-side web application logic. 

 TWO NEW SHOWBOAT TOOLS: CHARTROOM AND DATASETTE-SHOWBOAT (10 MINUTE
READ) [14] 

 Chartroom and datasette-showboat expand the capabilities of a CLI
tool, Showboat, which assists coding agents in creating Markdown
documents to demonstrate their code. datasette-showboat enables
Showboat's new remote publishing feature, allowing incremental updates
of documents to a Datasette instance for real-time viewing, addressing
the previous delay in seeing agent-generated content. Chartroom is a
CLI charting tool designed for coding agents to easily generate and
embed various chart types, such as bar, line, and scatter graphs, with
alt text directly into Showboat documents.
