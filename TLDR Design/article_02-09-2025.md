 DOING (AND DIRECTING) GREAT DESIGN REQUIRES DETAIL OBSESSION (5
MINUTE READ) [7] 

 Great design relies on identifying one key organizing detail that
makes everything else work, which is often a subtle element that
creates hierarchy and guides attention rather than obvious features
like large headlines or bright colors. Senior designers must maintain
a connection to craft details throughout their careers, as strategic
vision fails without precise execution of countless small decisions.
The best design leaders delegate execution while retaining their
ability to spot these crucial details and teach others to recognize
the keys that unlock successful design. 

 USER-ECOSYSTEM THINKING: AN ANTHROPOLOGIC APPROACH TO DESIGN (7
MINUTE READ) [8] 

 Traditional UX models view users as isolated individuals, but a new
user-ecosystem thinking approach considers interconnected roles and
relationships that affect product interactions. This framework
recognizes that successful design depends on entire networks of
people, from surgical teams using medical tools to various
stakeholders involved with smart doorbells. As AI and automation blur
boundaries between humans and technology, new concepts like "autonomic
users" and "active artifacts" better capture the complexity of modern
human-technology relationships.

 DESIGNING FOR DISCOVERY: WHY AI AND ACCESSIBILITY ARE NOW UX
PRIORITIES (4 MINUTE READ) [13] 

 Web designers now face a new challenge as AI agents join human users
in navigating websites, creating the need for Agent Experience (AX)
design that makes sites legible to both human and AI users. The
emerging field overlaps significantly with accessibility practices
since both AI agents and assistive technologies rely on semantic
structure, clear metadata, and proper HTML rather than visual
elements. Best practices include using semantic HTML tags,
implementing ARIA attributes and alt text, creating structured files
like llms.txt, and ensuring content isn't hidden in JavaScript to
future-proof sites for AI-driven search and autonomous agents. 

 OPTIMIZING PWAS FOR DIFFERENT DISPLAY MODES (10 MINUTE READ) [14] 

 Progressive Web Apps can introduce usability issues when
transitioning away from browser environments, as users lose access to
familiar browser functionalities such as back buttons, refresh
options, and URL sharing. PWAs support various display modes that can
be targeted using CSS media queries and JavaScript to customize the
user experience accordingly. Developers can use display-mode media
queries to hide installation prompts for existing PWA users, provide
alternative navigation controls, and tailor content to match the more
committed app-like usage context.
