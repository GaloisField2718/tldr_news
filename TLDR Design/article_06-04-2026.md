 LEAKER SAYS THERE'S NO BACK TO BLACK FOR THE IPHONE 18 PRO (1 MINUTE
READ) [4] 

 Apple dropped the black color option for the iPhone 17 Pro. It now
offers only blue, orange, and silver. Despite some user
disappointment, the bold Cosmic Orange proved highly popular,
especially in China, contributing to strong sales. A reliable leaker
now suggests the black finish still won't return for the iPhone 18
Pro, indicating Apple is sticking with more distinctive, status-driven
color choices. 

 CURSOR REFRESHES ITS VIBE CODING PLATFORM WITH FOCUS ON AI AGENTS (2
MINUTE READ) [5] 

 Cursor released Cursor 3, a new version of its AI coding platform
that focuses on AI agents to help developers automate programming
tasks. The platform features a chatbot interface where developers can
describe features in natural language and choose which AI model to
use, with both cloud-based and desktop agents working together. New
features include Design Mode for UI editing, multi-agent task
completion, and improved code review processes.

component libraries and usage guidelines so prototypes start from
production-aligned code rather than generic components. Make
attachments let designers supply actual files вЂ” such as PDFs,
datasets, and screenshots вЂ” so prototypes reflect real constraints
and content instead of idealized approximations.
