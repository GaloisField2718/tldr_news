 INTRODUCTION TO AT PROTOCOL (43 MINUTE READ) [4] 

 The Authenticated Transfer Protocol (ATProto) is the underlying
technology of Bluesky. This article describes the core components of
ATProto, including records, blobs, lexicons, DIDs, handles, and AT
URIs, outlining how they structure data and identify users. It also
goes over the server architecture, focusing on the roles of PDSes
(Personal Data Servers), relays, AppViews, labellers, and feed
generators in processing and distributing data across the network. 

 HOW I BUILT A FULL-STACK REACT FRAMEWORK 4X FASTER THAN NEXT.JS WITH
4X MORE THROUGHPUT (8 MINUTE READ) [5] 

 Rari is a full-stack React framework built on a custom Rust runtime
with V8. Its architecture consists of three layers: a Rust core
runtime for executing React Server Components, smart Vite integration
for RSC-aware transformations and dependency management, and true
server-side rendering with streaming support. It uses file-based
routing with automatic code splitting and prefetching, optimizing
bundle sizes and performance. 

 STATE MANAGEMENT ARCHITECTURE IN NX (4 MINUTE READ) [6] 

 Nx and NgRx Signal Store provide a structured approach to Angular
application state management. The architecture in this article goes
over a library-based structure with clear separation of concerns
between data access, state management, UI, and features. It enforces
dependency rules to prevent architectural drift and allows benefits
like build optimization, testing isolation, and team scalability.

 HERB (GITHUB REPO) [11]
