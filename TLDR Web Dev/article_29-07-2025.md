 THE USELESS USECALLBACK (5 MINUTE READ) [4] 

 `useCallback` and `useMemo` are often misused, leading to unnecessary
complexity and overhead without actual performance gains. This article
goes over scenarios where memoization is pointless, such as when
passing props to non-memoized components or using props directly as
dependencies within `useCallback` or `useEffect`.Â A real-world
example is shown from the Sentry codebase to illustrate how seemingly
memoized values can be invalidated by non-memoized dependencies. 

 A FRIENDLY INTRODUCTION TO SVG (21 MINUTE READ) [5] 

 SVG is an image format with XML syntax that allows for dynamic
manipulation with CSS and JavaScript. It offers UI primitives like
circles, rectangles, and polygons, and their attributes can be styled
and animated. This post explains presentational attributes like
`stroke` and `stroke-dashoffset`, which are useful for creating
animated effects like drawing shapes. A key concept explained is the
`viewBox` attribute, which creates an internal coordinate system that
allows SVGs to scale properly and remain responsive, solving the
problem of fixed positioning that occurs when using absolute
coordinates. 

 ADDING .MD URLS FOR RAW MARKDOWN CONTENT IN NEXT.JS (5 MINUTE READ)
[6] 

 It's possible to add ".md" URLs to a Next.js blog to serve raw
markdown content. This solution uses Next.js rewrites for efficient
URL mapping, directing requests for ".md" files to a dedicated API
route. The API route then retrieves the raw markdown content from
Content Collections and returns it with appropriate headers, including
caching instructions.

 FLOWGRAM (GITHUB REPO) [9] 

 FlowGram is a node-based flow building engine for creating workflows

 GITMCP (GITHUB REPO) [10] 

 GitMCP is an open-source Model Context Protocol (MCP) server designed
to eliminate code hallucinations in AI tools by providing them with
access to up-to-date documentation and code from GitHub projects. It
supports both specific repository and generic server access, allowing
AI assistants like Cursor to fetch relevant information.

 Since adopting Go in 2016, American Express has learned multiple

MINUTE READ) [12] 

 PyroWave is a fast, low-latency video codec designed for local game
streaming. This codec focuses on simplicity, removing motion
prediction and entropy coding for speed and error resilience, using
intra-only compression and discrete wavelet transforms. Benchmarks
show faster encoding and decoding speeds compared to standard codecs
like H.264/HEVC/AV1 under similar constrained conditions.
