ARTICLES & TUTORIALS

 LESS IS SAFER: HOW OBSIDIAN REDUCES THE RISK OF SUPPLY CHAIN ATTACKS
(3 MINUTE READ) [9] 

 Supply chain attacks involve malicious updates sneaking into
open-source code. Obsidian minimizes risk by depending on minimal
third-party code, often re-implementing functions or forking modules
when feasible. When using larger libraries, Obsidian uses version
pinning and lockfiles, guaranteeing a deterministic build process. 

 BRINGING OBSERVABILITY TO CLAUDE CODE: OPENTELEMETRY IN ACTION (16
MINUTE READ) [10] 

 Claude Code can be monitored using OpenTelemetry and SigNoz to gain
visibility into its usage and impact. By using OpenTelemetry for data
collection and SigNoz for visualization and alerting, teams can track
important metrics like token usage, cost, and performance trends. This
article shows how to set up OpenTelemetry instrumentation for Claude
Code. 

 CLAUDE CAN (SOMETIMES) PROVE IT (19 MINUTE READ) [11] 

 Claude Code is quite good at interactive theorem proving (ITP), a
difficult formal method used to verify critical systems. Claude Code
can complete complex proof steps independently, but still requires
human guidance for the overall formalization. This points towards a
future where ITP is more accessible and doesn't require expert
knowledge.

 APPLE'S THINNEST IPHONE STILL STANDS UP TO REPAIRS (11 MINUTE READ)
[18] 

 The iPhone Air, Apple's thinnest iPhone, surprisingly prioritizes
repairability through clever design choices. By shifting the logic
board and using a flat disassembly tree, the phone avoids stacking
components, making repairs easier. The battery is easily accessible
with electrically debonding adhesive, and the USB-C port is modular. 

 THE HIDDEN RISK IN NOTION 3.0 AI AGENTS: WEB SEARCH TOOL ABUSE FOR
DATA EXFILTRATION (6 MINUTE READ) [19] 

 CodeIntegrity discovered a vulnerability in Notion 3.0's AI Agents
related to its web search tool that allows for data exfiltration. By
embedding a malicious prompt within a seemingly harmless PDF,
attackers can trick the AI agent into querying a controlled server
with sensitive data extracted from the user's Notion pages. This
indirect prompt injection attack exploits the agent's tool access.
