 FARMERS INSURANCE DATA BREACH IMPACTS OVER 1 MILLION PEOPLE (2 MINUTE
READ) [6] 

 A data breach at Farmers Insurance, disclosed via a third-party
vendor, exposed personal information of over one million people,

directly hacked; the sensitive data was stolen from a compromised
external database. Authorities and affected individuals have been
notified.

 Docker patched a critical container escape flaw (CVSS 9.3) affecting
Windows and macOS. The vulnerability (CVE-2025-9074) allowed attackers
to access host files from within containers, posing major risks to
containerized environments.

 RAISING SECURITY WITH ORGANIZATION-WIDE YUBIKEY (FIDO2) IN ENTRA ID
(17 MINUTE READ) [10] 

 Hunt & Hackett details its end-to-end YubiKey FIDO2 rollout for
Microsoft Entra ID, using phased conditional access enforcement to
mandate phishing-resistant hardware authentication for specific users
and actions. Its approach includes automated Temporary Access Pass

exceptions, and robust monitoring of breakglass accounts. Key lessons
include session token control and cross-platform testing prior to full
enforcement.

prompts in GitHub issues. GitHub responded with security improvements
like enhanced confirmation flows, tool execution restrictions, and
sandboxed environments. The post also recommends Workspace Trust and
containerization as defenses, emphasizing the need for ongoing
protection as LLM threats evolve.

 PHISHING EMAILS ARE NOW AIMED AT USERS AND AI DEFENSES (6 MINUTE
READ) [17] 

 Phishing campaigns are now embedding AI prompt injection commands
within emails to manipulate automated analysis tools while
simultaneously targeting users with credential harvesting tactics.
These hidden prompts instruct AI models to enter reasoning loops
rather than flag malicious content, increasing the risk of delayed
detection or false negatives in SOC workflows. This dual-track
approach highlights attackers' growing awareness of AI-based defenses
and their efforts to bypass both human and machine detection systems.
