ATTACKS & VULNERABILITIES

 ERMAC V3.0 BANKING TROJAN SOURCE CODE LEAK EXPOSES FULL MALWARE
INFRASTRUCTURE (2 MINUTE READ) [6] 

 Complete source code for Android banking Trojan ERMAC 3.0 leaked from
an open directory, revealing a Malware as a Service (MaaS)
infrastructure targeting over 700 banking, shopping, and
cryptocurrency apps. The leak exposed critical security flaws,
including hardcoded JWT secrets, static admin tokens, and default
credentials in the malware's PHP/Laravel backend and React frontend.
ERMAC 3.0 features enhanced form injection methods, AES-CBC encrypted
communications, and a Kotlin-based Android backdoor that avoids
infecting devices in countries in the Commonwealth of Independent
States (CIS). 

 PLEX WARNS USERS TO PATCH SECURITY VULNERABILITY IMMEDIATELY (2
MINUTE READ) [7] 

 Plex has reached out to some of its users and urged them to update
their Plex Media Server installation due to a recently patched
security vulnerability. Plex has not assigned a CVE to the
vulnerability or released details yet, but has stated that it was
discovered via the Plex bug bounty program. 

 UK TELECOM FIRM COLT SUFFERS MASSIVE RANSOMWARE ATTACK (3 MINUTE
READ) [8] 

 The Warlock ransomware gang targeted the UK telecommunications firm
Colt, causing disruptions to its online and voice services. In
response, Colt took its systems offline, resulting in service
interruptions, and is currently in the process of restoring
operations. The attackers have claimed to have stolen sensitive data,
potentially exploiting a flaw in Microsoft SharePoint.

 POST-QUANTUM TLS IN PYTHON (5 MINUTE READ) [10] 

 In preparation for the creation of a Cryptographically Relevant
Quantum Computer (CRQC), NIST selected three post-quantum (PQ)
cryptographic standards. This post details how to test hybrid TLS,
where classical keys are used in combination with PQ ML-KEM keys, in
Python on AWS. boto3, the Python SDK for AWS, relies upon the Python
`ssl` module, which in turn relies on OpenSSL, which added support for
hybrid TLS in version 3.5. 

 KAFKA ENCRYPTION FOR CARDHOLDER DATA: SOLVING PCI CHALLENGES WITH
KROXYLICIOUS (10 MINUTE READ) [11] 

 This article explains how to achieve end-to-end encryption of credit
card data in Kafka without changing any application code, using the
Kroxylicious proxy. This solution simplifies PCI compliance and data
protection by encrypting messages transparently, minimizing disruption
to existing systems and workflows.

MISCELLANEOUS

 RUSSIAN GROUP ENCRYPTHUB EXPLOITS MSC EVILTWIN VULNERABILITY TO
DEPLOY FICKLE STEALER MALWARE (4 MINUTE READ) [16] 

 Russian threat actor EncryptHub is exploiting patched Windows
vulnerability CVE-2025-26633 via social engineering and malicious MSC
files to deploy Fickle Stealer malware and backdoors. The group uses
fake IT support requests, a compromised Brave Support platform, and
fake videoconferencing to gain control. Attacks feature encrypted C2,
persistence, and evasion techniques like fake browser traffic. 

 GOVERNMENT PAPERS FOUND IN AN ALASKAN HOTEL REVEAL NEW DETAILS OF
TRUMP-PUTIN SUMMIT (1 MINUTE READ) [17] 

 State Department documents with sensitive details about the August 15
Trump-Putin summit were left in an Alaskan hotel's printer. The eight
pages revealed meeting locations, times, and phone numbers of U.S.
government employees involved in the summit at Joint Base
Elmendorf-Richardson. Guests found the papers, photographed them, and
reported the breach. 

 HOW YOUR SOLAR ROOFTOP BECAME A NATIONAL SECURITY ISSUE (7 MINUTE
READ) [18] 

 Vulnerabilities in home solar inverters pose cybersecurity risks,
potentially exposing the power grid to attacks. Lax industry security,
regulatory gaps, and widespread adoption have made these devices a
national concern, highlighting the complexity of securing modern,
distributed energy networks
