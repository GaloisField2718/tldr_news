ATTACKS & VULNERABILITIES

 AUSTRALIAN IVF GIANT GENEA BREACHED BY TERMITE RANSOMWARE GANG (2
MINUTE READ) [5] 

 One of Australia's largest fertility service providers, Genea, has
reported a breach in which attackers stole 940.7GB of data. Gather
attackers obtained access through a Citrix server and then obtained
access to the company's primary file server, DC, backup program, and
BabySentry patient management system. The breached data includes full
names, email addresses, phone numbers, dates of birth, emergency
contacts, next of kin data, health insurance numbers, and medical
history. 

 PYPI PACKAGE WITH 100K INSTALLS PIRATED MUSIC FROM DEEZER FOR YEARS
(2 MINUTE READ) [6] 

 Socket has discovered a malicious Python package called automslc that
pirates music from the Deezer streaming service via hardcoded
credentials. The automslc package utilized user supplies credentials
to login to Deezer or hardcoded credentials and stores downloaded
files locally. Socket reported that the package has been downloaded
over 100K times.

STRATEGIES & TACTICS

 SECURING TOMORROW'S SOFTWARE: THE NEED FOR MEMORY SAFETY STANDARDS (5
MINUTE READ) [8] 

 This blog post, from Google's Security team, discusses the urgent
need to eliminate memory safety vulnerabilities in software to enhance
security and trust in technology. The authors propose a
standardization framework to promote the widespread adoption of
memory-safe practices and tools in the software industry. Google is
actively involved in advocating for memory safety standards,
developing potential standards with partners, and prioritizing
memory-safe languages to reduce vulnerabilities and enhance security
in its products and services. 

 THREE PASSWORD CRACKING TECHNIQUES AND HOW TO DEFEND AGAINST THEM (4
MINUTE READ) [9] 

 This post discusses brute force, dictionary, and rainbow table
attacks to crack passwords. To defend against these techniques, users
should choose strong, unique passwords and enable multi-factor
authentication. Tools like Specops Policy provide an extra layer of
protection by continuously scanning Active Directory against a
database of over 4 billion breached passwords. 

 JOB PROFILES OF CERTIFIED SOC ANALYST: ROLE, RESPONSIBILITIES, AND
CAREER PATH (4 MINUTE READ) [10] 

 This blog post explains the role of a SOC Analyst, which involves
monitoring and responding to cybersecurity threats within
organizations. Their progression requires technical skills with SIEM
tools and security systems.

LAUNCHES & TOOLS

 STREAMLINING VULNERABILITY RESEARCH WITH IDA PRO AND RUST (8 MINUTE
READ) [11] 

 Marco Ivaldi of HN Security has released new tools in Rust for
reverse engineering and vulnerability research in IDA Pro that
leverage the IDA SDK and Binarly's idalib Rust bindings for a faster
user experience. These tools, which include rhabdomancer and haruspex,
help locate potentially insecure API calls and extract pseudo-code for
analysis, enhancing the efficiency of vulnerability research efforts.
The availability of idalib opens up opportunities for developers to
create standalone analysis tools in Rust, expanding the capabilities
of IDA Pro in security research. 

 TURBO INTRUDER SCRIPTS (GITHUB REPO) [12] 

 A collection of Turbo Intruder scripts to emulate the 4 main types of
Burp Intruder attacks - battering ram, cluster bomb, pitchfork, and
sniper. 

 ECLIPSERECON (GITHUB REPO) [13] 

 EclipseRecon is a powerful and stealthy web reconnaissance tool
designed to uncover hidden vulnerabilities, subdomains, and intricate
site structures that may otherwise remain in the dark.

 OPENSSF RELEASES SECURITY BASELINE FOR OPEN SOURCE PROJECTS (2 MINUTE
READ) [14] 

 The Linux Foundation's Open Source Security Foundation (OpenSSF)
released a security baseline for open source projects to improve their
security and trustworthiness. This checklist outlines best practices
to reduce vulnerabilities and enhance project security levels.
Developers are encouraged to meet at least level 1 requirements, with
larger projects advised to aim for level 3.
