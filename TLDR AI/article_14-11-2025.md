HEADLINES & LAUNCHES

 AI STARTUP CURSOR RAISES $2.3 BILLION FUNDING ROUND AT $29.3 BILLION
VALUATION (3 MINUTE READ) [5] 

 Cursor has closed a $2.3 billion funding round at a $29.3 billion
post-money valuation. The tool has crossed $1 billion in annualized
revenue and swelled to more than 300 employees since it first launched
in 2023. Coding tools have proved to be a lucrative AI use case.
OpenAI approached Anysphere to potentially purchase Cursor earlier
this year, but the deal failed to gain traction. 

 MURATI'S THINKING MACHINES IN FUNDING TALKS AT $50 BILLION VALUE (3
MINUTE READ) [6] 

 Thinking Machines Lab is in early talks to raise new funding at a
roughly $50 billion valuation. The new financing would more than
quadruple the startup's valuation from July. The deal's terms have yet
to be finalized and could change. Thinking Machines aims to build AI
models and products to help businesses by supporting more 'human-AI
collaboration' across every field of work. The startup has dozens of
employees who previously worked at OpenAI. 

 GOOGLE ADDS DEEP RESEARCH IN NOTEBOOKLM (3 MINUTE READ) [7] 

 Google's latest update to NotebookLM introduces Deep Research, a
feature designed to support high-quality source integration, and adds
support for analyzing new file types, including spreadsheets and Word
documents.

 INTRODUCING GPT-5.1 FOR DEVELOPERS (2 MINUTE READ) [11] 

 GPT-5.1 uses fewer tokens on simple tasks and shows modest benchmark
gains on SWE-bench (+3.5 points to 76.3%) and GPQA Diamond (+2.4

or slightly lower. 

 INTRODUCING CODE WIKI: ACCELERATING YOUR CODE UNDERSTANDING (3 MINUTE
READ) [12] 

 Code Wiki is a platform that maintains a continuously updated,
structured wiki for code repositories. It scans the full code base and
regenerates documentation after each change, so docs evolve with the
code. Every wiki section and chat answer is hyperlinked directly to
the relevant code files and definitions. The platform will allow new
contributors to start immediately, while senior developers can now
understand new libraries in minutes.

making it easier to identify which neurons link together into a
â€œcircuitâ€ representing a certain concept. The approach
successfully isolated circuits for tasks like Python quote-matching
