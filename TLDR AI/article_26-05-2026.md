 ON-POLICY DISTILLATION (5 MINUTE READ) [14] 

 On-policy distillation trains a student model on trajectories sampled
from its own policy while a teacher provides dense token-level
supervision through KL-based regularization, closing the
train-inference distribution mismatch that off-policy methods suffer.
The canonical formulation unifies forward-KL, reverse-KL, and JSD
losses with reverse-KL emerging as the default for mode-seeking
smaller students, and a one-line code swap of the regularizer model on
top of an RL stack like Tinker implements the technique.

 INTRODUCING BENCHBENCH (5 MINUTE READ) [16] 

 BenchBench is a benchmark that tests how well models can create a
benchmark. It works as a great benchmark for model abilities as well
as a test of models' self-awareness. The benchmark tests creativity
and not just problem-solving ability. In tests, GPT 5.2 was the only
winner, with every other model, from Opus 4.6 to GPT 5.5, struggling
to create an actually useful benchmark that others had a hard time
solving.
