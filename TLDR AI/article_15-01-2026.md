 OPENAI SIGNS $10 BILLION DEAL FOR COMPUTE (1 MINUTE READ) [5] 

 OpenAI signed a multi-year deal with Cerebras worth over $10 billion
to receive 750 megawatts of compute through 2028. The partnership aims
to boost inference speed for OpenAI's products. 

 GEMINI INTRODUCES PERSONAL INTELLIGENCE (6 MINUTE READ) [6] 

 Gemini's Personal Intelligence beta, launching in the US, connects
Google apps to personalize user experiences by securely linking Gmail,
Photos, YouTube, and Search for tailored assistance. Users control app
connections, enhancing the AI's ability to provide specific, informed
responses. Privacy is prioritized, ensuring data remains secure and
under user control while continuously refining functionality. 

 AIRBNB POACHES FORMER META GENAI LEADER TO BE NEW TECHNOLOGY CHIEF (3
MINUTE READ) [7] 

 Airbnb appointed Ahmad Al-Dahle, former head of GenAI at Meta, as its
new technology chief to enhance AI in travel and e-commerce. The
company is shifting beyond short-term rentals, integrating new
services, AI chatbots, and aiming to become a travel concierge.
Al-Dahle's extensive experience with AI at Meta and Apple aligns with
Airbnb's strategic focus on tech-driven growth.

 UNDERSTANDING MANUS SANDBOX - YOUR CLOUD COMPUTER (7 MINUTE READ) [9]


 Manus Sandbox is a fully isolated cloud virtual machine that runs in
its own environment. Manus allocates a sandbox for each task and can
execute sandboxes in parallel. Sandboxes have full capabilities,
including networking, file systems, a browser, and various software
tools. They can work around the clock to complete assigned tasks
without consuming local resources. 

 SOFTWARE TOO CHEAP TO METER (6 MINUTE READ) [10] 

 AI coding agents are making software development nearly free,
allowing for personalized applications tailored to individual needs.
These agents, like Anthropic's Claude Code, can automate small
projects that save time. While the technology is still developing, it
promises rapid changes in software customization and productivity by
2026.

 HOW SHOPIFY AND GOOGLE BUILT THE UNIVERSAL COMMERCE PROTOCOL (UCP) SO
AI AGENTS CAN TRANSACT WITH ANY MERCHANT (SPONSOR) [11] 

 Shopify and Google co-developed an open standard for AI agents: the

 GPT-5.2-CODEX IS NOW AVAILABLE IN THE RESPONSES API (2 MINUTE READ)
[13] 

 GPT-5.2-Codex is an upgraded version of GPT-5.2 optimized for agentic
coding tasks in Codex and similar environments. It supports four
levels of reasoning effort settings and has a 400,000-token context
window. The model costs $1.75 per million input tokens and $14.00 per
million output tokens. It is now available in the Responses API. 

 TOOL SEARCH NOW IN CLAUDE CODE (2 MINUTE READ) [14] 

 MCP Tool Search is now rolling out in Claude Code. The feature allows
Claude Code to dynamically load tools into context. Claude Code
detects when MCP tool descriptions would use more than 10% of the
context, and when triggered, tools are loaded via search instead of
preloaded. MCP tools will otherwise work exactly as before. 

 MINISTRAL 3 TECHNICAL REPORT (18 MINUTE READ) [15] 

 Ministral 3 is a new family of dense language models (3B, 8B, and
14B) optimized for low-resource environments, offering base,
instruction-tuned, and reasoning variants. The models support image
understanding and were trained using Cascade Distillation, an
iterative distillation and pruning method.
