 OPENAI AND ORACLE REPORTEDLY INK HISTORIC CLOUD COMPUTING DEAL (2
MINUTE READ) [5] 

 OpenAI reportedly signed a $300 billion deal with Oracle for compute
power over five years, starting in 2027, marking one of the largest
cloud contracts ever. This move diversifies OpenAI's cloud providers
beyond Microsoft Azure, aligning with its involvement in the $500
billion Stargate Project for domestic data centers. Oracle has been
working with OpenAI since 2024, highlighting a strategic shift in
cloud resources essential for OpenAI's growth.

limit, testing and fixing its own code while building applications

 CLAUDE API: WEB FETCH TOOL (3 MINUTE READ) [12] 

 The new web fetch tool in the Claude API fetches content from URLs as
part of its response to prompts. It can extract full text content from
URLs and PDFs. For security, the tool can only fetch URLs that have
previously appeared in the conversation context, so it cannot fetch
arbitrary URLs that Claude generates or URLs from container-based
server tools. Anthropic recommends only using the tool in trusted
environments or when handling non-sensitive data, as enabling it poses
data exfiltration risks.

and scaling to tens of rounds. The full training pipeline is public.

 AI AS TELEPORTATION (5 MINUTE READ) [14] 

 While AI has the potential to improve our lives in a lot of ways, it
is important to be wary of secondary effects and unintended
